ID e CLASS

ID

O ID funciona como um CPF. Assim como só posso ter um numero de CPF, cada elemento só pode ter um valor ID. O CPF é o unico, ou seja, ninguem pode ter um CPF igual ao meu. Isso significa que o ID tambem é unico, entao não é possivel repetir o mesmo valor duas vezes.

CLASS

O CLASS funciona de maneira oposta ao ID. Diferente do ID em que você só pode ter um valor, no CLASS você pode ter varios valores. Tambem diferente do ID, em que nao pode repetir um valor, no CLASS você pode utilizar o mesmo valor para varios elementos.
